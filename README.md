# Prerak Patel - Portfolio

A modern, responsive portfolio website built with React, Vite, and a custom "Matrix/Cyber Security" aesthetic.

## Tech Stack
- **Framework:** React + Vite
- **Styling:** Vanilla CSS + CSS Variables (`src/index.css`)
- **Icons:** React Icons
- **Animations:** Framer Motion

## Features
- **Responsive Design:** Works on desktop and mobile.
- **Dark Theme:** High-contrast Green/Black theme tailored for developer aesthetics.
- **Animations:** Smooth scroll reveals and micro-interactions.
- **Sections:**
  - **Hero:** Typing effect introduction.
  - **About:** Professional bio and terminal-style "code" visual.
  - **Tech Stack:** Grid of skills with icons.
  - **Projects:** Featured projects with GitHub links.
  - **Contact:** Call to action.

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Customization
- **Theme:** Edit variables in `src/index.css` to change colors.
- **Content:** Update components in `src/components/` to change text or add new projects.
