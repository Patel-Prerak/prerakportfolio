# Prerak Patel - Portfolio

A modern, responsive portfolio website built with React, Vite, and a custom "Matrix/Cyber Security" aesthetic.

## Tech Stack
- **Framework:** React + Vite
- **Styling:** Vanilla CSS + CSS Variables (`src/index.css`)
- **Icons:** React Icons
- **Animations:** Framer Motion
- **Particles:** tsParticles

## Features
- **Responsive Design:** Works on desktop and mobile.
- **Dark Theme:** High-contrast Green/Black theme tailored for developer aesthetics.
- **Animations:** Smooth scroll reveals and micro-interactions.
- **SEO Optimized:** Comprehensive meta tags, Open Graph, Twitter Cards, and structured data.
- **Sections:**
  - **Hero:** Typing effect introduction with particle background.
  - **About:** Professional bio and terminal-style "code" visual.
  - **Tech Stack:** Grid of skills with icons.
  - **Projects:** Featured projects with GitHub links.
  - **Contact:** Call to action.

## SEO Features
- ✅ **Meta Tags:** Title, description, keywords, and author tags
- ✅ **Open Graph:** Facebook and social media sharing optimization
- ✅ **Twitter Cards:** Enhanced Twitter sharing with large image cards
- ✅ **Structured Data:** JSON-LD schema for better search engine understanding
- ✅ **Sitemap:** XML sitemap for search engine crawlers
- ✅ **Robots.txt:** Proper crawler directives
- ✅ **Canonical URLs:** Prevent duplicate content issues
- ✅ **Semantic HTML:** Proper heading hierarchy and HTML5 elements

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Deployment

This portfolio is deployed on Vercel. To deploy your own version:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel will auto-detect Vite and deploy

**Live URL:** [https://prerak-portfolio-b3pla962l-patelprerak435-1434s-projects.vercel.app/](https://prerak-portfolio-b3pla962l-patelprerak435-1434s-projects.vercel.app/)

## Customization
- **Theme:** Edit variables in `src/index.css` to change colors.
- **Content:** Update components in `src/components/` to change text or add new projects.
- **SEO:** Update meta tags in `index.html` and the SEO component.
- **Logo:** Replace `/public/logo-v2.svg` with your own logo.
